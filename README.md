# Aircraft Radar Simulator

**Aircraft Radar Simulator** is an interactive web-based simulation that visualizes multiple aircraft moving in real time on a dynamic radar screen. Each plane has unique properties including position, speed, altitude, and trajectory. The simulator highlights potential collisions and provides an intuitive, responsive display.

### Features

* **Real-time radar display** with aircraft represented as rectangles and labeled with call sign, altitude, and speed.
* **Dynamic trajectories**: each aircraft moves along a unique path with customizable speed and direction.
* **Collision detection**: planes at the same altitude turn red when paths overlap.
* **Responsive canvas**: adapts to any screen size, maintaining smooth animation.
* **Customizable simulation**: control total aircraft and maximum altitude using interactive sliders.
* **Randomized aircraft generation**: unique call signs and flight parameters for realistic simulation.

### Usage

1. Open `index.html` in any modern web browser.
2. Adjust **total aircraft** and **maximum altitude** using the sliders in the info panel.
3. Watch the planes move, detect collisions, and display flight information dynamically.
